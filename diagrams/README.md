# Diagrams

Mermaid source files for all workflow diagrams. Render them in any Mermaid-compatible viewer (GitHub renders them inline, or use [mermaid.live](https://mermaid.live)).

| File | What it shows |
|------|--------------|
| [git-workflow.mmd](git-workflow.mmd) | Full Git lifecycle from working directory to remote |
| [branching-strategy.mmd](branching-strategy.mmd) | GitFlow vs GitHub Flow side by side |
| [merge-vs-rebase.mmd](merge-vs-rebase.mmd) | Before/after for merge and rebase |
| [repository-permissions.mmd](repository-permissions.mmd) | GitHub role hierarchy |

All diagrams are also embedded inline in the relevant docs — these standalone files are here so you can adapt them for presentations.
