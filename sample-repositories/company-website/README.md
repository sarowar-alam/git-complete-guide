# company-website

A static HTML/CSS/JS company website. Use this for Git practice and as the primary repo for deployment in Lab 10.

## Run Locally

Open `index.html` directly in your browser — no server needed.

```bash
cd sample-repositories/company-website
# On Windows:
start index.html
# On any system with a browser:
open index.html   # macOS
xdg-open index.html  # Linux
```

## Structure

```
company-website/
├── index.html       # Home page
├── about.html       # About page
├── contact.html     # Contact page
├── styles/
│   └── main.css     # All styles
└── scripts/
    └── main.js      # Interactivity
```
