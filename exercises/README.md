# Exercises

Practice problems organized by difficulty. Work through beginner first, then move to intermediate and advanced when you're comfortable.

| Level | Exercises | Focus |
|-------|-----------|-------|
| [Beginner](beginner/) | 3 exercises | Init, commit, basic branching, simple merge |
| [Intermediate](intermediate/) | 3 exercises | Conflicts, rebase workflow, fork + PR |
| [Advanced](advanced/) | 3 exercises | Interactive rebase, cherry-pick, bisect |

Each exercise gives you a scenario and a set of tasks. Try to complete them without looking at the answers. Check your work in [answers/exercise-solutions/](../answers/exercise-solutions/).
